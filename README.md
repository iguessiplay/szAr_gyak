# Számítógép-architektúrák gyakorló oldal

## Indítás
Nyisd meg az `index.html` fájlt böngészőben.

## Bővítés
A `questions.js` fájlban tudsz új feladatokat hozzáadni.

Támogatott típusok:
- `truefalse`
- `matrix`
- `number`
- `text`
- `dramCommands`

A projekt teljesen statikus, nincs szükség Node-ra vagy szerverre.
